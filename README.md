# pyqt-single-image-graphics-view
PyQt one image only QGraphicsView

## Requirements
* PyQt5 >= 5.8

## Setup
`pip3 install pyqt-single-image-graphics-view`

## Method Overview
* `setFilename(filename: str)`
* `setPixmap(p: QPixmap)`
